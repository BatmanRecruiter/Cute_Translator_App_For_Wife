# Repository description

Mobile-first PWA for casual English ↔ Mexican Spanish text-message translation, built for Crystyl with automatic language detection, one-tap copy, iPhone Home Screen support, and a blue-and-gold sparkle theme.

# Portfolio description

Crystyl Translator is a personal-use Progressive Web App designed for bilingual text messaging. It automatically detects English or Spanish input, translates to the opposite language with a Mexican Spanish preference, and lets the user copy the result directly into iMessage. The app is intentionally lightweight: no login, no database, no saved message history, no build system, and no backend. It also includes iPhone Home Screen support, service-worker app-shell caching, and a custom quote-card experience that makes the utility feel personal instead of generic.
